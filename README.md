This is designed to be a nice chunk of boilerplate that sets up a NodeJS project using:

* ES6
* Webpack
* MongoDB
* SocketIO
* Travis CI
* Eslint
* CodeCov
* Heroku
* Jade
* Stylus

[You can see it live on Heroku here.](https://nodestack-.herokuapp.com/)

It uses Node 6.9.0 and Mongo 3.2

### Building

    nave usemain 6.9.0
    npm install
    npm run build
    npm start

    # Development Mode
    npm run watch

### Testing

    npm run test
